## 📚 What’s Inside?

This project covers:

- ✅ Variables, Constants, and Data Types  
- ✅ Functions and Return Values  
- ✅ Conditional Statements (if, switch)  
- ✅ Loops (for, range)  
- ✅ Arrays, Slices, and Maps  
- ✅ Structs and Methods  
- ✅ Error Handling  
- ✅ Goroutines and Channels (Basic Concurrency)
